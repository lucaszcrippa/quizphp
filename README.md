🎮 Quiz de Personalidade Geek

Um site simples desenvolvido em HTML, CSS e PHP que descobre qual tipo de geek você é através de um quiz divertido.

🧠 Sobre o Projeto

O Quiz de Personalidade Geek é um projeto web onde o usuário responde algumas perguntas relacionadas ao universo geek (tecnologia, jogos, criatividade, etc.).

Ao final do questionário, o sistema analisa as respostas e mostra qual perfil geek mais combina com a pessoa.

Exemplos de resultados possíveis:

👨‍💻 Programador

🎮 Gamer

🎨 Designer

🧠 Cientista

🏗️ Arquiteto

O objetivo do projeto é praticar conceitos básicos de desenvolvimento web, como formulários, estilização e lógica de programação.

🛠️ Tecnologias Utilizadas

HTML5 → estrutura do site

CSS3 → estilização e layout

PHP → lógica do quiz e cálculo do resultado

📂 Estrutura do Projeto
quiz-geek/
│
├── index.html        # Página inicial do quiz
├── style.css         # Estilo do site
├── resultado.php     # Processa as respostas e mostra o resultado
│
└── img/              # Pasta de imagens
    ├── fundo.jpg
    ├── programador.png
    ├── gamer.png
    ├── designer.png
    └── arquiteto.png
▶️ Como Executar o Projeto

Instale um servidor local (XAMPP, WAMP ou Laragon).

Coloque a pasta do projeto dentro da pasta htdocs.

Inicie o servidor Apache.

Abra o navegador e acesse:

http://localhost/quiz-geek

Responda as perguntas e descubra qual geek você é.
